# boekhoud.py

DOS-style accounting software in Python — terminal UI, fully offline, privacy-first.

Inspired by Exact voor DOS and Cash Software. Built as a personal tool, learning project, and portfolio piece.

## Status

🚧 Phase 1 in development — main menu and basic dispatch working.

## Stack

- Python 3.11+
- Textual (TUI framework)
- SQLite (+ SQLCipher, optional)
- ReportLab (PDF generation)
